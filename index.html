Summary: Latitude vs. X


For this project, weather patterns were analyzed in over 500 cities to see how distance from the equator played a role. After collecting an ample amount of data using an API, matplotlib was used to graph the different factors of weater vs. latitude.

Different factors include: temperature, cloudiness, wind speed and humidity. The purpose of this site is to provide the resulting graphs and the source data that was used to create aforementioned graphs. In addition, there is a short summary to go along with each graph. Lastly, there is a comparisons page with all graphs present on one screen.
